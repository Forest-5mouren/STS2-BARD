# 吟游诗人 Bard — Slay the Spire 2 Mod

基于 D&D 5e 规则的吟游诗人，为 *Slay the Spire 2* 增添了一位全新可玩角色。

依赖 [RitsuLib](https://github.com/BAKAOLC/STS2-RitsuLib) 制作。

## 总览

| 项目 | 数量 |
|------|------|
| 卡牌总数 | 83 张 |
| 遗物 | 4 个 |
| 关键词 | 3 个（魔法/乐曲/吟唱） |
| 学院分支 | 3 个学院 + 日常线 |

## 三大学院

### 🛡️ 勇气学院 — Vigor（活力）交互
围绕 **活力** 资源运作，消耗和累积活力来获得收益。

- **ValiantPresence**：失去活力时获得格挡
- **RhythmStorm**：使用乐曲/魔法获得活力，获得活力时造成伤害
- **ExplosivePower**：消耗活力来检索各类型卡牌
- **ValiantDash**：消耗活力来抽牌

### ⚔️ 剑舞学院 — Follow-up（上一张牌联动）
根据上一张打出卡牌的类型获得额外收益。

- **SwordDance**：上一张是攻击则多段，是技能则得格挡
- **Disengage**：上一张是攻击则抽牌，是技能则得格挡
- **FlurryThrust**：本回合打出过攻击/技能牌则追加伤害
- **HoneyedTongue**：上一张是技能则全体虚弱
- **FollowUpStrike**：攻击/技能联动
- **ParryRiposte**：上一张是技能则回能

### 🎭 轶闻学院 — Multiplayer（多人/buff）
面向多人模式的群体支援与群体削弱。

- **BansheeWail**（吟唱）：下回合全体敌人虚弱+易伤
- **InspiringMelody**（吟唱）：下回合全体队友力量+敏捷
- **SereneSong**（吟唱）：下回合全体队友回复生命
- **Allegro**：全体友方每回合获得能量
- **BriskBeat**：全体友方每回合抽牌

### 📜 日常线 — 谱曲/演奏
诗人的日常活动——谱曲、练习、即兴表演。

- **Improvise**：每回合随机获得一首乐曲（排除吟唱牌）
- **AllNightComposition**：获得能量，代价是晕眩
- **Encore**：根据本场战斗已打出的乐曲牌数量造成伤害
- **StreetPerformance**：本回合演奏过乐曲则获得金币
- **EchoingMelody**：每回合复制上一张乐曲牌

## 关键词系统

| 关键词 | 效果 |
|--------|------|
| **魔法** (Magic) | 吟游诗人吹奏乐曲来引动魔法力量 |
| **乐曲** (Song) | 吟游诗人所谱写的小曲，不需要魔法，仅凭声音就能发挥音乐的力量 |
| **吟唱** (Chant) | 吟唱卡牌的效果不会立即触发，改为在下一回合开始时自动生效。不会因受到伤害而中断。 |

## 和声系统

**和声**是诗人的核心计数器，由初始遗物 **诗人激励** 提供：

- 每打出一张**魔法牌**或**乐曲牌**，和声 +1
- 回合开始时，全体友方获得和声层数的**活力**，然后清空（升级遗物调整为 2）
- 多张卡牌与和声联动：**RhythmStrike**（和声>3获得能量+抽牌）、**DancingYouth**（和声>3下回合多抽牌）、**ArcaneEcho**（和声达阈值时回收法术）

## 遗物

| 遗物 | 效果 |
|------|------|
| **诗人激励**（初始） | 每打出魔法/乐曲牌，和声+1。回合开始全体获得对应活力，和声清零 |
| **诗人激励+**（升级） | 同上，但和声调为 2（不清零） |
| **唤蛇长笛** | 第一回合将蛇咬+加入手牌，使其获得乐曲 |
| **2226的小提琴** | 演奏乐曲时，给与敌方全体 2 层灾厄 |
